# Pulse Health Club — Portfolio Site

A modern, responsive single-page website for a fictional UK health club. Built with **React**, **TypeScript**, and **Vite**, with animations via **Framer Motion**.

## Features

- **Hero** — Full-viewport hero with gradient background, stats, and CTAs
- **Features** — Why Pulse (equipment, coaching, community, flexible membership)
- **Classes** — Class types (HIIT, Strength, Yoga, Boxing, Spin, Pilates)
- **Testimonials** — Carousel of member quotes
- **Pricing** — Three tiers (Flex, Commit, Elite) with highlighted “most popular”
- **Contact** — Address, hours, and contact form with success state
- **Sticky header** — Transparent → solid on scroll, mobile hamburger menu
- **Smooth scroll** — In-page navigation and scroll behaviour
- **Animations** — Staggered section and card animations, testimonial transitions

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 19 + TypeScript
- Vite 7
- Framer Motion
- CSS (custom design system, no UI library)

Design: dark theme, Syne + DM Sans, copper/teal accent palette.
